# Free tier and plan enforcement

Organisation plans (FREE / PRO / ENTERPRISE) with enforceable limits: teams, mock server count, mock request rate per day, lint report retention, and optional mock TTL for inactive servers. Spectral custom ruleset persistence moves from in-memory stub to database (Pro).

## Status

<StatusBadge status="planned" /> <PriorityBadge priority="P0" />

## Free tier (target)

- Up to 5 teams per org; unlimited API specs
- Limited mock servers and daily mock call quota
- Built-in Spectral rulesets; custom rulesets Pro
- MCP and CLI available on free tier to drive adoption
