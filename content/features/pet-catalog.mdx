# Pet Catalog

Browse and search pets by category, breed, and price. The main discovery experience for the pet store.

## Status

<StatusBadge status="in-progress" /> <PriorityBadge priority="P0" />

## Description

The Pet Catalog allows customers to:

- Browse pets by category (dogs, cats, birds, etc.)
- Search by breed, name, or price range
- View pet details, images, and availability
- Add pets to cart directly from the listing

## Completion

~60% — Listing and basic search done. Image gallery and filters in progress.
