# Hosted MCP server

Per-organisation MCP endpoint (e.g. `https://mcp.winspect.io/org/{orgId}/mcp`) exposing tools such as `list_apis`, `search_apis`, `get_api_spec`, and `get_mock_url`, authenticated with the org API key. Enables Cursor, Claude Desktop, and Windsurf to query the API catalog using natural language.
