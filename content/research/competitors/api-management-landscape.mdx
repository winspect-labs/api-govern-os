# API Management Landscape

## Summary

Kong, Apigee, AWS API Gateway, and Backstage dominate different segments. Winspect positions as the governance layer—unified inventory, subscription-based authorization, and AI-powered discovery—for teams that want governance without replacing gateways or auth.

## Competitor Overview

| Competitor | Segment | Strength | Weakness |
|------------|---------|----------|----------|
| Kong | API gateway | Plugins, ecosystem | Gateway-centric; governance secondary |
| Apigee | Enterprise | Analytics, scale | Heavy, complex, expensive |
| AWS API Gateway | Cloud | AWS integration | Vendor lock-in; governance limited |
| Backstage | Developer portal | Catalog, plugins | Manual catalog; no authz enforcement |

## Winspect Positioning

Governance-first. Works with any gateway. Subscription-based authorization on top of existing auth. K8s discovery and RAG search for discoverability. Agent-accessible product knowledge.
